# Python Async Comprehension

Async generators and comprehensions in Python.

## Files

- `0-async_generator.py` - `async_generator` yields 10 random numbers
- `1-async_comprehension.py` - `async_comprehension` collects values using async comprehension
- `2-measure_runtime.py` - `measure_runtime` runs 4 comprehensions in parallel

## Requirements

Python 3.7+
