# Pagination

Python pagination implementations.

## Files

- `0-simple_helper_function.py` - `index_range` function
- `1-simple_pagination.py` - Simple pagination with `get_page`
- `2-hypermedia_pagination.py` - Hypermedia pagination with `get_hyper`
- `3-hypermedia_del_pagination.py` - Deletion-resilient pagination

## Requirements

Python 3.7+
