# Caching

Implementation of various caching algorithms in Python.

## Files

- `base_caching.py` - Base class with MAX_ITEMS = 4
- `0-basic_cache.py` - BasicCache (no limit)
- `1-fifo_cache.py` - FIFOCache (First In First Out)
- `2-lifo_cache.py` - LIFOCache (Last In First Out)
- `3-lru_cache.py` - LRUCache (Least Recently Used)
- `4-mru_cache.py` - MRUCache (Most Recently Used)

## Requirements

Python 3.7+
